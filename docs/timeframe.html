OUTLINE OF THE PROJECT CHARTER


Stage 1:

The first stage of our team project is to conduct our first meeting and determine which path we want to take in regards to development.

Stage 2:

The cloud has been created so we can all collaborate and work efficiently.

Stage 3:

Browsing the web for alternatives and learning what is involved in creating the bot via reverse engineering.



Stage 4:

Our programmers will be working on and coding the discord bot to create a beta version that will be used later on for debugging.

Stage 5:

Team members will be documenting our project within the cloud shared workspaces well as conducting further research on how to improve the beta version.

Stage 6:

The project will be tested live on the AWS hosted cloud in real time within our discord server to find and fix potential bugs.



Stage 7:

Our associated documentation will be formatted for submission before our deadline and added to our project website.

Stage 8:

Reflecting on the resources used and comparing our estimated time of completion to the actual time.

Stage 9:

The complete version of the bot will be released and the associated documentation will be submitted to the project administrator (Anthony C).
